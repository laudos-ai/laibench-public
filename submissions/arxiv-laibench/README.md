# Beyond Templates arXiv draft package

This package contains a draft manuscript and support material for an arXiv-style preprint on radiology report variability.

## Files

- `main.tex`: manuscript source.
- `references.bib`: bibliography.
- `main.pdf`: compiled manuscript preview used by the frontend.
- `metadata.json`: title, authors, abstract, suggested category, and submission blockers.
- `arxiv-laibench-source.zip`: arXiv source upload package containing `main.tex`, `references.bib`, and `00README`.
