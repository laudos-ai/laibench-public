laibench pro

The controlled radiology report benchmark.

LAIBench Pro tests a narrow, auditable task: given an exam descriptor and text findings, can a system produce a faithful radiology report without losing clinically important facts?

The pt-BR controlled suite is gated. The public page shows methodology and aggregate diagnostics only; case JSON, answer keys, frozen predictions and corpus provenance stay outside the public repository.

Built by Laudos.AI. Reproducibility is available to authorized reviewers inside the private audit environment.

laibench pro / Methods

Scope

LAIBench Pro is a controlled technical benchmark framework — not a medical device, not regulatory approval, not clinical validation. The harness evaluates report generation from provided text evidence; it is not primary image interpretation. The pt-BR controlled suite, corpus provenance, frozen predictions and answer keys remain gated outside the public repository. Authorized reviewers can reproduce runs inside the controlled adjudication environment.

laibench pro / API

Laudos.AI evaluation access

This public page is not a generic harness tutorial and does not publish private API routes, keys, prompts, frozen predictions, or command-line recipes for third-party systems.

What the benchmark represents

The controlled pt-BR suite now contains 120 staged cases, including critical-safety and negative-control strata. The displayed Laudos.AI row is the validated 120-case production-agent run. Runs are locked to a suite hash, scoring mode, track, scaffold class, and provenance chain before they are shown here.

Reviewer workflow

Authorized reviewers can reproduce the evaluation inside the controlled benchmark environment. Public visitors should read this page as a methodology and leaderboard surface for Laudos.AI, not as integration documentation for unrelated models or endpoints.

Product API

Operational access to Laudos.AI is handled through the product onboarding and permissioning flow. Benchmark artifacts and product credentials are intentionally separated.
